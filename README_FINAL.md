# Smart Water Quality System

Final README for the Smart Water Quality System project. This file summarizes the project, features, installation, usage, and API details.

## Project Summary

A polished FastAPI application for classifying water quality using sensor data, batch uploads, analytic dashboards, and report generation.

## How to Install

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## How to Run

```powershell
python app.py
```

## Main URLs

- Dashboard: `http://127.0.0.1:8000/dashboard`
- Statistics: `http://127.0.0.1:8000/statistics`
- Swagger docs: `http://127.0.0.1:8000/docs`
